# Teaspoons Array Imputation Pipeline Overview

## Purpose
We want an internal doc we can refer to for a refresher on the Array Imputation (also known as the Beagle) pipeline.  This will describe the pipeline inputs/outputs as well as pointers to all the different wdls involved in the pipeline.

## Stairway Flight
Information about the stairway flight used to launch this pipeline is [here](wdl_based_pipelines.MD)

## Pipeline Versions

### v2
#### Changelog
This version of the pipeline has two major differences from version 1.  The first being that hom ref sites in the imputed output were removed from the variant vcf and put into a new sites only vcf output.  This made the total output size much smaller.  The other major difference is that the reference panel used for this pipeline has been left aligned so that the variants in the output files are standardized to what our users would expect.  The underlying WDL versioning started at [3.0.0](https://github.com/broadinstitute/warp/releases/tag/ImputationBeagle_v3.0.0) and is currently the latest version in our service

#### Wdls
* Quota Consumed - https://github.com/broadinstitute/warp/blob/ImputationBeagle_v3.0.0/pipelines/wdl/arrays/imputation_beagle/ArrayImputationQuotaConsumed.wdl
* Input QC -  https://github.com/broadinstitute/warp/blob/ImputationBeagle_v3.0.0/pipelines/wdl/arrays/imputation_beagle/input_qc/ArrayImputationQC.wdl
* Pipeline - https://github.com/broadinstitute/warp/blob/ImputationBeagle_v3.0.0/pipelines/wdl/arrays/imputation_beagle/ImputationBeagle.wdl

#### Pipeline Inputs
* multiSampleVcf
  * Required
  * This is a multisample vcf that contains variant data for array samples the user wants to impute
* outputBasename
  * Required
  * Basename for outputs generated by pipeline
* minDr2ForInclusion
  * Optional
  * Default: 0.0
  * This will filter out any variants from the outputs whose DR2 value is below the number provided

#### Pipeline Outputs
* imputedMultiSampleVcf
  * A multi-sample VCF file containing imputed variant genotypes for all samples
* imputedMultiSampleVcfIndex
  * The index file for imputedMultiSampleVcf
* imputedHomRefSitesOnlyVcf
  * A sites only VCF file containing information about homozygous reference sites
* imputedHomRefSitesOnlyVcfIndex
  * The index file for imputedHomRefSitesOnlyVcfIndex
* contigsInfo
  * A TSV file containing metrics information about the input contigs (chromosomes)
* chunksInfo
  * A TSV file containing QC information about the chunks used during imputation
